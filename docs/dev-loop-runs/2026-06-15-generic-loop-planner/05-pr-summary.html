Summary: Evolva now supports a generic natural-language Loop Engineering flow. Users can type /loop <需求>, review a generated Loop Draft, revise it, confirm it, and only then execute or save the generated LoopSpec.

What Changed

  • Added LLM-first intent-to-loop planner with sanitized heuristic fallback.
  • Added draft lifecycle commands in CLI and TUI.
  • Added persisted active draft state.
  • Added strict validation for generated specs.
  • Added runtime enforcement for execution budgets.
  • Documented the user-facing workflow in README.

User-visible Behavior

/loop 做一个响应式 landing page，有 hero、pricing、FAQ
/loop revise 增加移动端验收
/loop confirm
/loop execute
/loop save landing-page-loop

The first command does not modify code. It creates a draft with phases, checkpoints, risks, command allowlist, and execution limits.

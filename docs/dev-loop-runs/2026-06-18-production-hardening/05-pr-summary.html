Residual Risks

The default backend remains local for developer compatibility; Docker isolation is available by configuration and can be validated with evolva sandbox smoke, but CI still does not run against a live Docker daemon. Pattern-based redaction should continue to grow with project-specific secret detectors. Prometheus text export is available; push-based OpenTelemetry or hosted logging adapters remain deployment-specific. CI uses scoped mypy checks for the newly hardened primitives; full-repo typing remains a migration item.
